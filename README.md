## Creating the Video

1. Open `index.html` in a browser
2. Use screen recording to capture 5-6 seconds of the animation
3. Save as `thanksgiving-policyengine-silent.mp4`
4. Run `python3 mix_soundtrack.py` to generate and combine audio

## Requirements

For soundtrack generation:
```bash
pip install numpy scipy
```

FFmpeg is required for audio/video processing.
